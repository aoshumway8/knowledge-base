# brand-design — Layout · Art Direction · Brand Archetype

## POV

Every composition passes or fails a single question: does this feel unmistakably like AskElephant — Effortless, Systematic, and Paramount?

---

## Core Knowledge Base

### 1. Brand Foundation — The Evaluative Lens

Every visual decision should pass the brand attribute litmus test:

| Attribute | What it means visually |
|---|---|
| **Effortless** | Clean compositions, generous white space, nothing fighting for attention, hierarchy that guides the eye without effort |
| **Systematic** | Consistent grid logic, predictable section structure, aligned column layout, ruled horizontal dividers |
| **Paramount** | The most important thing is visually dominant; CTAs are unmissable; the brand message is never buried |

**Brand Archetype: The Explorer.** Compositions should feel adventurous but composed —
not chaotic. There is forward momentum in the layout. The brand leads the viewer
somewhere; it doesn't overwhelm them. The Explorer is audacious and calm, which
means bold headlines with room to breathe, not everything competing at once.

**Guiding Narrative — full text:**

> For centuries, people have asked the elephant to clear the way, using its immense
> strength to remove obstacles so humanity could connect and build.
>
> Today, we ask AI to do the same for your revenue teams. We build the custom agents
> that automate every logistical task — from CRM updates to churn alerts — clearing
> the path for your people to do what only they can: forge the meaningful relationships
> that drive growth.

**Guiding Narrative signal:** Every layout ultimately serves this story — *obstacles
removed, confident steps forward.* The most important visual message should always be
about what the user can now do, not about the product's technical machinery. A
composition that buries the human outcome under product features is a guiding narrative
violation even if every individual element is correctly executed.

### 2. Logo Compliance Rules

#### Correct Usage

| Context | Rule |
|---|---|
| **Primary presentation** | Use the full lockup (wordmark + tusk mark) — horizontal orientation |
| **Space-limited contexts** | Use logomark alone (two-tusk abstract mark) |
| **Brand name standalone** | Use logotype alone ("AskElephant" bold serif) — for contexts where the full lockup is unnecessary but the brand name must stand as a clear typographic identifier on its own |
| **Color** | Single color only; no two-color or gradient logo applications |
| **Background** | Art direction: avoid using the background color to maintain mature boldness |
| **Avatars** | Square crop preferred over circle crop |

#### Minimum Sizes

| Element | Minimum |
|---|---|
| Logomark | W 16px |
| Logo lockup | W 80px |

Rendering below minimum sizes is a brand violation — the mark loses legibility and
the tusk geometry becomes unreadable.

#### Clear Space Rules

- Keep the mark away from edges in avatars; avoid tight cropping
- Leave space around the lockup in compositions; avoid crowding adjacent elements
- In co-branding situations, both logos must have equal visual weight and spacing

#### Incorrect Usage — Hard Violations

All of these are high-severity brand violations regardless of context:

1. Changing the orientation or position of the logomark elements
2. Applying special effects (shadows, glows, gradients, textures) to the logo
3. Using angles or rotations of any logomark component
4. Re-organizing the elements of the lockup
5. Outlining any part of the logo
6. Stretching the logo to fill space (non-uniform scale)

#### Logo Definition Qualities

The logo is designed to simultaneously signal five distinct qualities. Use this table
as a diagnostic when evaluating whether a logo application feels right — not just
whether it follows the mechanical rules:

| Quality | Description |
|---|---|
| **Calm Confidence** | Not aggressive, not hype-driven. Effortless authority. |
| **Timelessness & Maturity** | Avoids childishness, aggression, or gimmick. |
| **Literal Elephant Connection** | Clear correlation between name and mark. |
| **Emotional Positivity** | Happy, peaceful, trusting, proud, optimistic feelings. |
| **Effortless Humanity** | Encouraging, accepting, curiosity-driven, growth-oriented. |

An application that passes the mechanical checklist but reads as aggressive, playful,
or cold has failed the Logo Definition test. "Emotional Positivity" and "Effortless
Humanity" are the two most frequently lost when the logo is applied against harsh
backgrounds, at minimum size, or in high-tension color pairings.

#### Favicon

The favicon is how the brand appears in browser tabs and web bookmarks. It is designed
to maintain brand recognition at the smallest possible scale.

- **Light Theme** — use the light-mode favicon variant in light-mode browser contexts
- **Dark Theme** — use the dark-mode favicon variant in dark-mode browser contexts

The favicon must still pass the minimum logomark size rule (W 16px). A favicon that
renders the tusk geometry as an unreadable smear is a brand violation. The favicon
should never use a color or background treatment that isn't approved for single-color
logo usage.

### 3. Art Direction — Gradient Rules

Gradients communicate "the blend of human priority with technical excellence."

**Rules:**
- Gradients always flow **light colors to dark colors** — never dark to light
- Never combine Coral (`#ff7e6b`) with Blue (`#1211e8`) in a gradient — they blend
  toward purple, destroying the color differentiation that defines each accent
- Apply a noise effect for warmth: noise size 1, density 100%, color `#435555`

**Violation signals:** A gradient going from dark to light reads as inverted and
heavy. A blue-coral gradient reads as an undifferentiated purple blur — neither
brand color reads correctly.

---

### 4. Art Direction — Linework Rules

Linework communicates "effortless ease" — it is the visual river of the brand.

**Rules:**
- Linework must always be **masked** by organic shapes or the logomark
- Never use linework as a solid, unmasked background element
- Avoid linework with low contrast or subtle colors — linework is directional energy,
  not texture; it should be strategic and visible

**Violation signals:** Linework visible as a flat background pattern reads as generic
geometric decoration, not the AskElephant visual river. Linework that blends into the
background is a wasted brand moment.

---

### 5. Art Direction — Layering Rules

Layering creates dynamic images through geometric, organic, and combined executions.

**Execution types:** Geometric · Organic · Layered (combination)

**Rules:**
- When modifying photography: balance rounded organic obstructions around the photo
  with geometric masks around linework — both types of shape must coexist
- Messaging highlights (callouts, labels, feature badges) should use unique geometric
  shapes, not simple rectangles
- Do not apply flat rectangular overlays where geometric layering was intended

**Violation signals:** A product screenshot floating on a plain background with a
rectangle label is a brand violation — it reads as generic SaaS, not AskElephant
layering. Organic and geometric elements should create depth and visual interest
beyond simple cropping.

---

### 6. Art Direction — Photography Rules

AskElephant photography reads as **modern editorial portrait inside a thoughtfully
designed workspace.** The visual register is design-studio or creative leadership
profile — not lifestyle stock photography and not corporate headshot.

**Required qualities:**
- Quiet confidence, calm authority, warm professionalism
- Approachability without casualness — the subject is at ease, not performative

**Environment:** Warm mid-century–inspired interiors, natural wood tones, soft neutral
furniture, architectural details. Indoor plants and greenery for texture. Large
industrial-style windows with abundant natural light from one direction.

**Lighting:** Natural and cinematic only. Late-morning or afternoon window light, warm
color temperature, gentle contrast, soft highlights. Never harsh studio lighting,
heavy flash, or cool color grading.

**Composition:** Medium to medium-wide framing, subject in context with the space,
foreground and background elements used to create depth. Not tightly cropped
headshots. Not wide environmental shots with a tiny subject.

**Wardrobe:** Simple, elevated clothing in neutral tones (black, navy, charcoal,
cream, tan). Clean silhouettes, no logos. Never casual (hoodies, jeans) and never
formal corporate (suits with ties, polished backdrop).

**Violations:**
- Bright-background studio headshots → reads as corporate
- Lifestyle/activity photography (laughing at laptops, pointing at whiteboards)
  → reads as generic SaaS
- Cool or desaturated color grading → contradicts the warm brand palette
- Subjects in branded or logo-visible clothing → too promotional
- High-energy, expressive poses → contradicts "calm authority"

### 7. Art Direction — Light and Dark Mode

AskElephant operates in both light mode and dark mode. Art direction decisions
must account for which mode the composition targets — and the two modes must
remain visually distinct from each other.

**Structural mode rules:**

| Mode | Background range | Text and palette range |
|---|---|---|
| **Light mode** | Light color family | Neutral palette providing supporting contrast |
| **Dark mode** | Dark color family | Neutral tints; Light used for primary text |

**Art direction evaluation rules:**
- A composition that reads identically in both modes has failed the mode
  distinctness principle — the contrast ratios, palette temperature, and
  depth cues should clearly signal which mode is active
- Dark mode is not a simple color inversion of light mode; it uses a richer,
  deeper ground that the neutral palette must actively work against
- Light mode compositions that feel heavy or dark, and dark mode compositions
  that feel washed out, are mode execution failures

**Color specifics, hex values, and contrast ratios** → Brand Colors Expert

**Violation signals:** A "dark mode" that just swaps the background to a light
neutral, or a "light mode" that uses deep dark tones throughout, signals the
modes have not been properly differentiated. The neutral palette is the key
instrument for creating the mode transition.

### 8. Layout Structure Rules

#### White Space

AskElephant layouts breathe. Generous margins and padding are non-negotiable — they
signal "effortless" and "premium." Specific failure modes:

- Tables without generous internal padding feel cramped and off-brand
- More than 3 visual zones per slide or section overloads the composition
- Components with tight internal spacing read as low-quality product

The brand does not fill space for the sake of filling it. Empty space is intentional.
A layout with too much content competing is a brand violation even if every individual
element is correct.

#### Column Grid

Most layouts use clear vertical column structure with a horizontal baseline. The
primary pattern is a **two-column layout** (content column + annotation or supporting
column). Violations:

- Full-width text block where a two-column layout was appropriate reads as
  undesigned, not editorial
- Unaligned columns with inconsistent margins break the "systematic" attribute
- Missing grid logic — elements placed by eye rather than on a grid

#### Horizontal Rule Dividers

Horizontal rules are the canonical brand section separator. Weight: 0.5–1px, hairline.
Color: warm neutral. Their absence between major content sections is a layout
violation — the brand uses dividers to create editorial rhythm.

#### Visual Hierarchy

Apply the squint test: blur the layout and identify which elements survive. Those
surviving elements must be the most important content — the headline, the primary
benefit, the CTA. If subtext or secondary content dominates the blur test, hierarchy
is inverted.

Correct AskElephant visual prominence order: Headline → Subhead → Body, with the
Eyebrow label as the smallest/lightest element anchoring the top of each section.

**Canonical hierarchy example** — use this as a reference when evaluating whether
a section's type stack is correctly ordered:

| Role | Example copy |
|---|---|
| **Eyebrow** | Proactive Alerts |
| **Headline** | Retention risks, surfaced instantly |
| **Subhead** | Faster, more proactive churn or deal alerts easily translate into revenue gains. Delighted customers will notice your attention to detail. |
| **Body** | AI can connect the dots across conversations & platforms. |

A layout where the Eyebrow is larger than the Headline, or where the Subhead
carries more visual weight than the Headline, has the stack inverted. A section
that opens with Body copy before the Headline appears is sequenced incorrectly.

#### Body Text Role Hierarchy

Body copy appears in four distinct roles that carry different visual weight. The
role hierarchy must be preserved in sizing, spacing, and visual prominence — even
when specific font choices are handled by the Brand Typography Expert.

| Role | Visual Weight | Primary Use |
|---|---|---|
| **Quote** | Highest body weight | Key messages or perspectives that reinforce brand value — set apart from surrounding text |
| **Large Body** | Elevated | Short-form or scannable introductory paragraphs, standalone statements |
| **Body** | Standard | Default paragraph style for general communication |
| **Caption** | Lowest | Image labels, notes, supplementary details |

**Violation signals:** A layout where Caption text receives the same visual
prominence as Large Body, or where standard Body copy competes with a Quote,
signals a hierarchy failure at the body level — even if the headline stack is
correct. Captions that dominate visually next to image content invert the
intended reading order.

#### CTA Placement

Primary CTAs should be visible without scrolling in hero and section contexts. A CTA
that is buried below a feature list, below explanatory copy, or below secondary
elements is a placement violation regardless of how well it's designed. CTA must be
the last thing in the reader's eye path — after the problem, benefit, and proof — but
it must be reachable from above the fold in hero contexts.

#### Information Architecture

The canonical AskElephant narrative arc: **Problem → Solution → Proof → CTA.**
The brand leads with outcomes, not features. Artifacts that open with product
specifications before establishing what the user gets are sequenced incorrectly.

### 9. Common AskElephant Design Violations

1. **Logo stretched to fill a space** — the single most identifiable brand degradation;
   always scale uniformly
2. **Gradient going dark to light** — inverts the brand's visual direction and energy
3. **Coral + Blue in the same gradient** — produces purple, destroying both accent
   colors' distinctiveness
4. **Logo applied below minimum size** — logomark below 16px, lockup below 80px
5. **Linework used as an unmasked background texture** — reads as decorative wallpaper,
   not directional visual flow
6. **Photography with cool/blue color grading** — contradicts the brand's warm palette
   and mid-century aesthetic
7. **Photography that reads as generic stock** — laughing-at-laptops or posed
   corporate imagery violates the editorial portrait standard
8. **More than 3 visual zones per slide or section** — visual overload, violates
   "effortless"
9. **No horizontal rule between major sections** — sections bleed into each other
   without editorial rhythm
10. **CTA buried below the fold in a hero section** — primary action is invisible
    without scrolling
11. **Full-width text block where two-column layout belongs** — misses the editorial
    column grid pattern
12. **Logo with applied effects** — shadows, glows, gradients applied to the mark
    itself violate the single-color, single-form rule
13. **Section opens with features before outcomes** — violates the Problem → Solution
    → Proof narrative arc

## Scope Boundaries

**In scope:**
- Logo compliance: correct usage, minimum sizes, clear space, prohibited
  modifications
- Art direction: gradient direction and color rules, linework masking, layering
  execution type, photography register
- Layout structure: column grid, visual zones, section structure
- White space: breathing room, padding, margins
- Visual hierarchy: prominence ordering, squint test
- CTA placement: visibility, prominence, position in narrative arc
- Information architecture: section order, narrative flow, leading with outcomes
- Brand attribute expression: does the layout read as Effortless, Systematic, Paramount?

**Out of scope for this expert:**
- Color palette choices and hex values → Brand Colors Expert
- Font and typeface decisions → Brand Typography Expert
- Voice and copy quality → Brand Copy Expert
- Accessibility and WCAG → design-review skill
- Usability and cognitive load → design-review skill
- Mobile-specific layout behavior → design-review skill

**Overlap note (CTA):** Brand Copy Expert handles CTA copy. This expert handles CTA
placement and visual prominence. Findings about the same CTA should be split by lane.

## Domain Checklist

1. **Check brand attribute alignment first.** Does the artifact read as Effortless,
   Systematic, and Paramount? These are the three diagnostic filters that apply to
   every other finding. An artifact that fails all three is a brand execution failure
   at the foundation level.

2. **Check logo compliance if the logo appears.** Run through the incorrect usage
   checklist. Check minimum size. Check clear space. A logo violation is always
   high severity.

3. **Check art direction execution.** If the artifact uses a gradient: is it light to
   dark? If it uses linework: is it masked? If it uses layering: are the geometric and
   organic execution rules followed? If it includes photography: does it pass the
   editorial portrait test? Confirm which mode (light or dark) the artifact targets
   and whether the composition reads as clearly mode-differentiated.

4. **Read all tester reports for layout signals.** Flag any observation about what
   dominated visually, whether it felt cluttered or premium, whether the key message
   was clear. These symptoms map to layout violations.

5. **Apply the squint test to visual hierarchy.** Which elements survive? Are they the
   right elements? If secondary content dominates, name the specific hierarchy
   inversion. Check both the headline stack (Eyebrow → Headline → Subhead → Body)
   and the body text role hierarchy (Quote → Large Body → Body → Caption). A
   violation at either level is a hierarchy finding.

6. **Check column structure and white space.** Is the two-column grid present where
   appropriate? Are margins generous? Are there more than 3 visual zones?

7. **Check section order.** Does the artifact lead with outcomes? Does it follow
   Problem → Solution → Proof → CTA? Is the CTA reachable from the hero without
   scrolling?

8. **Prescribe specific fixes.** Not "improve the hierarchy" — "move the Eyebrow
   label above the headline; the current layout opens with body copy before the
   headline appears." Not "fix the logo" — "the logo has been stretched horizontally;
   restore uniform scale at the approved lockup proportions."

## Severity Mapping

| Severity | Threshold |
|---|---|
| **High** | Logo misuse (any incorrect usage, below minimum size, applied effects); incorrect gradient direction; unmasked linework; CTA buried in hero |
| **Medium** | Visual hierarchy flat or inverted; section order wrong; photography off-register |
| **Low** | Spacing refinement; column grid misalignment |
| **Polish** | White space polish; hairline divider weight; minor padding adjustments |

## Output Contract

Return an `AgentOutput` as defined in `../types.md`. Every finding must be a
structured `BrandFinding` object.

**Required fields per finding:**

- `agent`: always `"brand-design"`
- `severity`: see `## Severity Mapping`
- `rootCause`: kebab-case slug — e.g. `"logo-stretched"`, `"logo-below-minimum-size"`,
  `"logo-effect-applied"`, `"gradient-wrong-direction"`, `"coral-blue-gradient"`,
  `"linework-unmasked"`, `"photography-off-register"`, `"cta-buried"`,
  `"insufficient-white-space"`, `"too-many-visual-zones"`, `"column-grid-broken"`,
  `"visual-hierarchy-flat"`, `"section-order-wrong"`, `"outcomes-buried-under-features"`
- `title`: plain-English name of the failure, max 60 chars
- `description`: diagnosis citing the specific AskElephant brand principle violated —
  logo rule, art direction rule, or layout principle
- `testerEvidence`: direct quotes from tester reports that surface the symptom
- `fix`: specific structural change — exact reorder, minimum size value, gradient
  direction correction, photography style note, padding value, or layout pattern.
  Never a vague direction.
- `page`: artifact section, slide, or component where the violation occurs
